Personal logo and portfolio website, modified/created for the Fall 2017 class of ART 608 Intermediate Web Design at Kansas State University.

Website Objectives

Overall this website was to give me an online presence and the experience of creating a website from the ground up.

  • The site was to match the personality of my personal logo.
  • I wanted it to have some unique element to it, to make the website stand out.
  • Overall simple website that isn't afraid of white.
  • Programmed in React to 1) gain experience with it 2) have planned future functionality that is easy to implement with React.

The Website

  • The website was made with the colors and technological aspects of my personal logo.
  • The circuit drawn onto a canvas element in the background was added to make the website unique.
  • A large header section was made on the home page to create some simplicity when first entering the website.
  • React framework was used throughout the site.
