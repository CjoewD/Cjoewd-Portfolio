A type experiment using the Pet brand evaporated milk packaging, created for the Fall 2016 class of ART 300 Experimental Type at Kansas State University.

Requirements

The project was based upon using all elements of the original packaging to create a design.

  • To start with we vectorized all of the type and artwork from the packaging.
  • The designs must include all elements of the original design: type, images, lines, etc.. However, you could hide them or make them visually irrelevant.
  • Initial designs were done in black and white. Later color was added and was used in proportion to the ratio of color on the packaging.
  • The design must feel like the brand it originated from.

Style Choices

  • I left mine in a rough state to keep a rustic feel to it, similar to the original packaging.
  • The lines were utilized for adding order and structure to an otherwise organic style.
