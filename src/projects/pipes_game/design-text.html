A simple plumbing game template in HTML5, created for the Fall 2016 class of CIS 580 at Kansas State University.

Design Requirements

  • The game world is divided into a regular grid, in which tiles representing pipes can be placed.
  • At the start of each level, an un-editable starting pipe and ending pipe appear.
  • At the start of the level, 'fluid' begins to flow from the start pipe. If the fluid manages to reach the end of assembled pipes, the game is lost.

Style Choices

  • Most of the pipes were provided by open source location. Thin straight piece, large cross section, and starting/ending pipes modified by me.
  • Dirt tiles were from open source locations.
  • The user interface and water is made from Javascript code.
  • The instruction page was made using Photoshop.
