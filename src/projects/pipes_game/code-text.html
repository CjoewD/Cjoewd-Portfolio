A simple plumbing game template in HTML5, created for the Fall 2016 class of CIS 580 at Kansas State University.

Requirements

For this assignment, you will be creating a clone of Pipe Mania, which will need to meet the following requirements:

  • The game world is divided into a regular grid, in which tiles representing pipes can be placed.
  • At the start of each level, an un-editable starting pipe and ending pipe appear. If the player manages to connect these with pipes, the level is completed. The start and end connection of the pipes should be available.
  • At the start of the level, 'fluid' begins to flow from the start pipe. If the fluid manages to reach the end of assembled pipes, the game is lost.
  • The game score and level are represented on-screen.
  • The kind of pipe available appear in random order, and the player can place them by left-clicking on the board.
  • Right-clicking on a pipe that does not yet have fluid in it rotates the pipe tile. Pipes with fluid in them cannot be rotated.
  • The speed at which fluid fills pipes increases as the player advances in level.
  • Placing pipes, rotating pipes, finishing a level, and loosing the game should all have sound effects that play when these events happen.
  • The game should play background music.
  • Instructions for playing the game should appear in-game or on the page.

Optional Requirements

  • Add at least 3 additional kinds of pipes with special attributes (i.e. a reservoir that fills up more slowly).
